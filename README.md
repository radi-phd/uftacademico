# Indicadores Acadêmicos UFT

Dashboard desenvolvido em Streamlit para análise de indicadores acadêmicos da UFT.

## Arquivos necessários

- indicadores2_uft.py
- 16.11.03a.ods
- requirements.txt

## Execução local

```bash
pip install -r requirements.txt
streamlit run indicadores2_uft.py
```

## Deploy no Streamlit Community Cloud

1. Envie todos os arquivos para o GitHub.
2. Acesse https://share.streamlit.io
3. Conecte sua conta GitHub.
4. Selecione o repositório.
5. Informe como arquivo principal:

```text
indicadores2_uft.py
```

6. Clique em Deploy.
